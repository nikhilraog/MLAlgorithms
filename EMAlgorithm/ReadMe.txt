Steps to run the EM Alogorith:

* Download the EM_MainDriver.java file
* Compile the code using the command
	javac EM_MainDriver.java
* Run the program using the below format
  EM <path/to/datafile/datafile.txt> <K: Number of clusters> <Initialization type 1 or 2><Emtype : 1 or 2> <Variance type : 1 - Non-Uniform variance OR 2 - variance of 1>

Example command:
C:\Users\nikhil\Desktop\em_data.txt 3 1 2 2
